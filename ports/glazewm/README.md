# Potter for GlazeWM

Themed window borders for [GlazeWM](https://github.com/glzr-io/glazewm), the
tiling window manager for Windows. Ships all three flavors: **Parchment**
(light), **Quill** (dark, default), **Ink** (deepest dark).

| File | What it is |
|------|------------|
| `potter-quill.yaml` / `potter-parchment.yaml` / `potter-ink.yaml` | A `window_effects` block — focused/unfocused border colors. |

## Apply it

GlazeWM has no `@import`/theme mechanism, so merge the block into your config by
hand. Open `%userprofile%/.glzr/glazewm/config.yaml`, find the existing
`window_effects:` section, and replace it with the contents of the flavor you
want (e.g. `potter-quill.yaml`). Then reload GlazeWM (`alt+shift+r` by default).

The focused border is Potter `peach` (`#e08a6a` in Quill); unfocused windows use
a muted overlay so the active window stays obvious.
