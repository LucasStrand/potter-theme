# Potter for Zebar

A drop-in stylesheet for [Zebar](https://github.com/glzr-io/zebar), the
customizable status bar / widget host for Windows (pairs with GlazeWM). Ships
all three flavors: **Parchment** (light), **Quill** (dark, default), **Ink**
(deepest dark).

| File | What it is |
|------|------------|
| `potter-quill.css` / `potter-parchment.css` / `potter-ink.css` | `--potter-*` custom properties + themed styles for the stock widgets. |

## Apply it

Zebar widgets are styled with CSS. Drop the flavor you want next to your widget
and import it from the widget's `style.css`:

```css
@import "potter-quill.css";
```

…or paste its contents into the window's `global_styles` in your Zebar config.

Beyond the `--potter-*` variables, it themes the vanilla widgets out of the box:
workspace buttons (focused → `peach`), binding-mode / tiling-direction chips,
the CPU high-usage warning (`red`), and the battery charging icon (`green`).
Reach for the variables to theme anything else, e.g. `color: var(--potter-text)`.
