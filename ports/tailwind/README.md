# @potter/tailwind

[Potter](https://potter.nu) for Tailwind — v4 `@theme` tokens + a v3 preset, alpha-aware.

```bash
npm i @potter/tailwind @potter/css
```

Tailwind v4:
```css
@import "@potter/css";
@import "@potter/tailwind";  /* bg-potter-base, text-potter-text, border-potter-overlay0 */
```

Tailwind v3: add `require("@potter/tailwind/preset")` to your `presets`. MIT.
