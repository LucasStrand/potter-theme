# @potternu/palette

The Potter color palette — source of truth for the [Potter](https://potter.nu) theme.
3 flavors (Parchment · Quill · Ink) × 26 colors, as JSON plus typed JS/TS bindings.

```bash
npm i @potternu/palette
```

```ts
import { palette } from "@potternu/palette";
palette.quill.colors.peach.hex; // "#e08a6a"
```

Or grab the raw data: `@potternu/palette/palette.json`. MIT.
