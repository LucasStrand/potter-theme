# @potternu/tailwind

[Potter](https://potter.nu) for Tailwind — v4 `@theme` tokens + a v3 preset, alpha-aware.

```bash
npm i @potternu/tailwind @potternu/css
```

Tailwind v4:
```css
@import "@potternu/css";
@import "@potternu/tailwind";  /* bg-potter-base, text-potter-text, border-potter-overlay0 */
```

Tailwind v3: add `require("@potternu/tailwind/preset")` to your `presets`. MIT.
